AuraLogic
AuraLogic is an interactive study app which aims to boost study logic with the use of flashcards to improve readiness in identifying the difference between life hacks and urban myths.
Purpose
The app is meant to create a fast an efficient way to help users study whilst receiving scores and pointers to the questions asked so that they are motivated to remember and try to improve.
Design considerations
Keeping things simple we used black and white as our interface,that allows the user to fully focus and function whilst using the app to maintaining their attention on the questions asked, instead of other things drifting their focus.
With the use of drawables and design constraints within the code a sleek interactive design is maintained throughout the app.
Features
The welcome screen provides an engaging interface which braces the user for the quiz up ahead.
The app consists of a flashcard system which constantly grabs the users attention because they are kept behind each other on screen as if you were holding them in your hand, keeping them wondering what the next question is.
The answer system is a True and False system, with the answer revealing itself after one of the two options is chosen using boolean statements.
Providing immediate responses on completion of a question, either being wrong or right and stating that it's either correct "keep going" or incorrect "that's an "urban myth".
A score sheet at the end of the quiz to show the user how their performance was with the questions being asked throughout the time.
Tools used

Andorid Studio: As the compiler for the foundation of the code

Behind the App
Through the use of  boolean statements validating  whether the answer is correct or incorrect and through the count function it tallies the score of correct and incorrect answers to create the score sheet which can be found in the end of the quiz for the user to see their progress.
